# 🌦️ Climate Weather App

A modern React-based weather application that provides real-time climate information for any city using the OpenWeather API.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature display
- 💧 Humidity and air pressure
- 🌬️ Wind speed and direction
- ☁️ Cloud coverage and weather condition
- 🎥 Video background UI
- ⌨️ Enter key support for quick search
- ⚡ Fast and responsive interface

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- CSS3
- OpenWeather API

##API Used
- OpenWeather API  
https://api.openweathermap.org
